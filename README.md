# Prueba Técnica QA Automation
**Postulante:** Jeniffer Ximena - Ingeniera en TI
**Fecha:** Febrero 2026

## Descripción del Proyecto
Este repositorio contiene la automatización de pruebas para la evaluación técnica de QA, cubriendo los siguientes flujos:
1. **E2E (Frontend):** Flujo de compra completo en [SauceDemo](https://www.saucedemo.com/).
2. **API (Backend):** CRUD de usuarios (Creaciòn, Escribir, Actualizar, Eliminar) en [PetStore Swagger](https://petstore.swagger.io/).

## Stack Tecnológico
* **Framework:** Cypress (v12.17.4 o superior)
* **Lenguaje:** JavaScript
* **Entorno:** Ejecutado en macOS Catalina / Cypress -> Electron Browser

## Pre-requisitos
Para ejecutar este proyecto se requiere tener instalado:
* [Node.js](https://nodejs.org/) (v14 o superior)
* npm (Gestor de paquetes)

## Instalación
Clonar el proyecto o descomprimir el archivo, y ejecutar en la terminal:
```bash
